# Unworthy QoL Mods

Quality of life mods for Unworthy, a hardcore combat-focused metroidvania without jumping developed by Aleksandar Kuzmanovic.

- Before installing any of these mods, it is recommended for you to experience the game as intended for the first playthrough
- Take note that the mods may not be entirely stable and unknown bugs may occur, reach out for any replicable bugs occurring

## Installation Guide

These mods were developed for Unworthy Version 1.0.1d

- Only tested on Windows, application of these mods on Mac / Linux via Proton or Wine not tested
- Currently only compatible with the GOG mirror of the game

How to install:

1. Install [DeltaPatcher](https://github.com/marco-calautti/DeltaPatcher/releases)
1. Download the `.xdelta` file for the desired mod
1. Find the install folder for Unworthy<br>
   - (GOG default at: `C:\Program Files (x86)\GOG Galaxy\Games\Unworthy`)
1. Make a copy of `data.win`, otherwise it will be overridden<br>
   - May keep the copy in the same directory, as long as there is a valid `data.win`
1. Run `DeltaPatcher.exe`
1. Drag `data.win` or enter it's file path into "Original file:"
1. Drag `***.xdelta` or enter it's file path into "XDelta Patch:"
1. Click "Apply patch"

Unworthy has now been modded and you may launch the game as normal

## Always Materialize

- The materialize ability gained by picking up the Crow's Eye relic after killing Father Amandil is now always active

## Faster Lifts

- Movement speed on ladders and platforms increased, improving map traversal
